# Suno Hotspot Composer

本地视频热点编辑器，用来把一段视频画面上的 20 个区域变成可点击按钮。

## 使用方式

需要通过本地后端打开，这样选满 4 个热点后才能调用生成流程。

```bash
pip install -r requirements.txt
uvicorn web_server:app --host 127.0.0.1 --port 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```

1. 打开页面后先在封面页点击“选择视频”，页面会停在视频第一帧。
2. 封面页会显示已配置的 20 个热点，可以直接点击 4 个热点选择音频。
3. 选满 4 个热点后，热点会渐隐，视频会在浏览器窗口内铺满播放，并自动调用后端生成歌名、歌词和歌曲。
4. 生成完成后，视频会停回第一帧，页面会播放生成好的音频并弹出二维码。
5. 关闭二维码弹窗后，页面会清空选择并进入下一轮。
4. 点击“编辑热点”进入热点编辑页面。
5. 在“编辑”模式里拖动椭圆，右下角圆点缩放，顶部圆点旋转。
6. 在右侧给每个热点改名字和音频文件。
7. 编辑页也可以切到“选择”模式，点击 4 个热点。
8. 右侧“程序输出”会得到 4 段音频的 JSON。

二维码默认会优先使用 OSS 或临时托管链接。如果要让手机直接访问本机服务，可设置 `PUBLIC_BASE_URL` 为这台电脑在局域网里的地址，例如：

```env
PUBLIC_BASE_URL=http://192.168.1.10:8000
```

热点配置保存在浏览器 localStorage，也可以用“导出配置”保存成 JSON。
